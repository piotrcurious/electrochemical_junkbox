# Small-Scale Electrochemical Reactors: Solar-Compatible, High-Value Output

Criteria applied: buildable with off-the-shelf materials, variable-current tolerant (solar-safe), impurity-resilient chemistry, and strong output value/cost ratio.

---

## Tier 1 — Excellent ROI, Very Buildable

### 1. Sodium Hypochlorite (Bleach) Generator
| | |
|---|---|
| **Inputs** | NaCl brine (seawater works) |
| **Outputs** | NaOCl solution (5–15% bleach), H₂ byproduct |
| **Electrodes** | Ti/MMO (dimensionally stable anodes) — long-lived, no coating erosion |
| **Cell voltage** | 3–4 V |
| **Impurity tolerance** | ★★★★★ — seawater-grade salt is fine; Mg²⁺, Ca²⁺ cause minor scaling only |
| **Solar suitability** | ★★★★★ — current directly scales NaOCl production rate; idle at night with μA trickle |
| **Market** | Disinfectant, pool sanitation, water treatment. Sells at €1–4/L for concentrated solution |
| **Notes** | Undivided flow cell is simplest. Keep temperature <40°C to avoid chlorate side-product. Ti/MMO anodes available cheaply from Chinese suppliers. Probably the single best option for off-grid chemical production. |

---

### 2. Alkaline Water Electrolysis (H₂ + O₂)
| | |
|---|---|
| **Inputs** | KOH or NaOH solution (20–30%) |
| **Outputs** | H₂ (fuel/industrial), O₂ |
| **Electrodes** | Nickel mesh/foam (cathode + anode) — very cheap |
| **Cell voltage** | 1.8–2.2 V |
| **Impurity tolerance** | ★★★★☆ — alkaline environment self-cleans many contaminants; avoid heavy metals in feedwater |
| **Solar suitability** | ★★★★☆ — production halts when current drops; electrolyte concentration stable; idle fine. Avoid letting cells dry |
| **Market** | H₂ for fuel cells, welding gas, lab use. O₂ for medical/aquaculture. H₂ purity ~99.5% achievable with simple drying |
| **Notes** | Much more impurity-tolerant than PEM. Ni electrodes can be made from nickel-plated steel or recovered Ni scrap. Stack design (bipolar) keeps footprint small. |

---

### 3. Copper Electrowinning from Leachate
| | |
|---|---|
| **Inputs** | CuSO₄ leachate (from PCB scrap, copper ore acid leach, wire waste) |
| **Outputs** | 99.9%+ pure copper cathode |
| **Electrodes** | Stainless steel cathode (reusable), lead-antimony or DSA anode |
| **Cell voltage** | 1.8–2.5 V |
| **Impurity tolerance** | ★★★★☆ — Fe²⁺, Ni²⁺, Zn²⁺ mostly stay in solution; Cu deposits preferentially at moderate current density |
| **Solar suitability** | ★★★★★ — slow deposition at low current just means longer run time; no harm in stopping/starting |
| **Market** | Copper at ~€8–10/kg, but the real value is in turning waste streams (PCB etch waste, acid mine drainage) into product. 10 kg/month realistic from small cell. |
| **Notes** | Harvest cycle: strip cathode deposit, replate. Leachate replenishment from raw scrap + H₂SO₄ is easy. Excellent circular economy angle. |

---

### 4. Silver Electrorecovery
| | |
|---|---|
| **Inputs** | Photographic fixer waste, X-ray developer, e-waste acid leachate, silver nitrate solution |
| **Outputs** | High-purity silver metal (>99%) |
| **Electrodes** | Stainless cathode, graphite or silver anode |
| **Cell voltage** | 0.5–1.5 V (very low!) |
| **Impurity tolerance** | ★★★★☆ — Ag deposits at lower voltage than most other metals; selectivity is inherent |
| **Solar suitability** | ★★★★★ — ultra-low voltage, can run directly from small PV with no controller |
| **Market** | Silver at ~€900/kg. Even small concentrations (1–5 g/L) in waste streams become viable. Photographic waste often free to acquire. |
| **Notes** | One of the highest value/watt ratios of any electrochemical process. A single recovered troy ounce (~31g, ~€28) can justify a full week's operation. |

## Tier 2 — High Value, Slightly More Demanding

### 5. Potassium Ferrate (K₂FeO₄) Production
| | |
|---|---|
| **Inputs** | KOH concentrated solution (10M+), iron anode |
| **Outputs** | K₂FeO₄ — a powerful oxidant, disinfectant, and water treatment agent |
| **Electrodes** | Pure iron/mild steel anode (consumable), nickel or stainless cathode |
| **Cell voltage** | 2–4 V |
| **Impurity tolerance** | ★★★☆☆ — KOH purity matters more here; works with technical-grade |
| **Solar suitability** | ★★★☆☆ — ferrate is unstable; must precipitate immediately with KOH; batch process fits solar well |
| **Market** | €50–200/kg as specialty oxidant/disinfectant. Water treatment companies pay premium. Niche but real market. |
| **Notes** | Ferrate is "super-bleach" — oxidizes micropollutants, pharmaceuticals, heavy metals simultaneously. Batch process: electrolyze → add KOH to precipitate purple K₂FeO₄ salt → filter → dry. |

---

### 6. Hydrogen Peroxide (H₂O₂) via Cathodic O₂ Reduction
| | |
|---|---|
| **Inputs** | Dilute H₂SO₄ or NaOH, O₂ (from air cathode or sparging) |
| **Outputs** | 3–15% H₂O₂ solution |
| **Electrodes** | Carbon black / graphite cathode (O₂ reduction), DSA or graphite anode |
| **Cell voltage** | 2–3 V |
| **Impurity tolerance** | ★★★☆☆ — transition metal ions catalyze H₂O₂ decomposition; avoid iron in electrolyte |
| **Solar suitability** | ★★★★☆ — production scales with current; dilute H₂O₂ is stable during idle |
| **Market** | Disinfectant, bleaching, crop treatment. 3–6% solution sells at €3–8/L in small quantities. |
| **Notes** | Gas diffusion electrode (GDE) construction is the main skill required — carbon black pressed into porous carbon paper. Achievable with basic materials. Avoids the hazardous Cl₂ chemistry of bleach. |

---

### 7. Electrochemical Ozone Generation
| | |
|---|---|
| **Inputs** | Deionized or tap water + small H₂SO₄ additive |
| **Outputs** | Ozone-enriched water (2–10 mg/L O₃) for disinfection/sterilization |
| **Electrodes** | Boron-doped diamond (BDD) or PbO₂ anode — BDD modules now affordable (~€30–80) |
| **Cell voltage** | 3–5 V |
| **Impurity tolerance** | ★★★☆☆ — BDD is chemically inert; mineral scale on electrodes is the main issue |
| **Solar suitability** | ★★★★☆ — ozone decomposes quickly so production-on-demand matches solar intermittency perfectly |
| **Market** | Ozonated water for food sanitation, bottling, aquaculture, dental. Sell the *unit* (€200–800) rather than the product. |
| **Notes** | BDD electrode modules are the key enabling component. Once sourced, the rest is PVC plumbing. Ozone concentration proportional to current density — very solar-linear behavior. |

### 8. Colloidal Silver Generator
| | |
|---|---|
| **Inputs** | Distilled water only |
| **Outputs** | 5–50 ppm colloidal silver solution |
| **Electrodes** | 99.99% silver rods/plates (both anode and cathode) |
| **Cell voltage** | 9–30 V, very low current (1–50 mA) |
| **Impurity tolerance** | ★★★★★ — feedwater just needs to be distilled; no salt or acid required |
| **Solar suitability** | ★★★★★ — trivially low power; a single small PV cell suffices |
| **Market** | Antimicrobial wound dressings, veterinary, alternative health market. Retail €15–50 per 250 mL. Extremely high margin. Regulatory grey area varies by country. |
| **Notes** | Lowest barrier to entry of any process here. Stir during electrolysis (magnetic stirrer) to keep particle size uniform. Particle size/concentration verified with laser pointer (Tyndall effect). |

---

### 9. Ammonium Persulfate / Peroxodisulfate
| | |
|---|---|
| **Inputs** | Ammonium sulfate solution (fertilizer grade — cheap, widely available) |
| **Outputs** | Ammonium persulfate ((NH₄)₂S₂O₈) — PCB etchant, polymerization initiator |
| **Electrodes** | Platinum-coated titanium anode (or smooth graphite with lower yield), stainless cathode |
| **Cell voltage** | 3–6 V |
| **Impurity tolerance** | ★★★☆☆ — Cl⁻ contamination causes chlorine side-products; use sulfate-only feedstock |
| **Solar suitability** | ★★★★☆ — batch process, pause/resume fine, product stable in solution |
| **Market** | PCB etchant in electronics. Industrial oxidant. ~€5–15/kg commercially; small-batch premium pricing possible for hobby/lab market. |
| **Notes** | High current density (300–500 mA/cm²) preferred for good Faradaic efficiency, so cell design matters. Electrode cost is the main barrier — Pt/Ti anodes ~€50–150. |

## Practical Solar Integration Notes

All of these handle variable current well, but with different strategies:

- **Batch processes** (ferrate, persulfate, colloidal Ag): Run during daylight, stop at night — product accumulates in solution.
- **Continuous flow processes** (bleach, ozone): Flow rate scales with current. During low irradiance, just slow the pump.
- **Electrowinning** (Cu, Ag): Deposition simply slows. No minimum current threshold needed.
- **H₂ generation**: Use a small buffer tank. Keep electrolyte warm to reduce activation overpotential on cloudy days.

A **MPPT controller** (€15–30) between PV panel and cell is recommended for all of them — it maximizes power extraction and prevents reverse current at night. Alternatively, a simple blocking diode + capacitor buffer suffices for low-current cells like colloidal silver.
